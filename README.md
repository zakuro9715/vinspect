# vv

Cli tool to develop vlang itself

## Commands

- [ ] inspect: inspect code
    - [ ] ast   : Print ast
    - [ ] tokens: Print tokens
- [ ] ci: CI tests
    - [x] cleancode
    - [x] fixed

## Installation

```
git clone https://github.com/zakuro9715/vv.git vv && cd vv
v . -prod
ln -s "$(pwd)/vv" $YOUR_PATH/vv
vv
```
